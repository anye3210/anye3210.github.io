前言

在上一篇文章中介绍了Redux在React Native中的基本使用，这篇文章介绍一下如何在Redux中自定义中间件。

中间件功能

中间件的作用主要是拦截指定的Action，进行自定义操作后继续执行该Action或指派执行其他Action。这里拦截的Action主要是指同步的Action，异步Action在Thunk中间件已经提供了相关功能。
